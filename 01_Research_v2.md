## 1. エグゼクティブサマリー

本リサーチ文書は「**IF記載要領2018（2019年更新版）**」および「**IF作成の手引き 令和2年5月改訂版**」に完全準拠したIF AI自動生成システム構築に向けた技術調査・要件分析を目的とする。

CTDはModule 1〜5から構成され、日本語・英語混在テキスト、表、数式、グラフ画像が複雑に混在する。テキストと関連する表・グラフ間の参照関係を高精度に維持しながらIF文書を生成することがシステムの核心課題である。

| 項目 | 内容 |
|------|------|
| 規格基準 | IF記載要領2018（2019年更新版）／IF作成の手引き令和2年5月改訂版 |
| CTD参照基準 | ICH CTD構成ガイドライン（PMDA文書 000156304） |
| IF構成 | 表紙＋目次＋略語表＋本編（Ⅰ〜ⅩⅢ）＋参照頁 |
| AI技術基盤 | ローカルLLM（Ollama）＋マルチモーダルRAG（テキスト・表・図統合） |
| 最終目標 | 人間作成IFとの文章・表・グラフ参照の高精度一致 |
| 稼働環境 | ローカルPC（GPU VRAM 16GB以上推奨）— インターネット不要 |

## 4. CTD文書の技術的特性と精密RAG構築の必要性

### 4.1 CTD文書の複合的なデータ形態

| データ形態 | 出現頻度 | 処理技術要件 | 誤抽出リスク |
|-----------|---------|------------|------------|
| 日本語テキスト | 非常に高 | SudachiPy形態素解析 | — |
| 英語テキスト | 高 | 英語トークナイザ | — |
| 日英混在文 | 高 | 言語切り替え検出 | 誤分割リスク |
| 数値・単位 | 非常に高 | 単位正規化・数値抽出NER | **単位変換ミスが致命的** |
| 構造化表（格子型） | 高 | Camelot（lattice） | 列ヘッダと値の対応ミス |
| 非格子型表（罫線なし） | 中 | pdfplumber（stream）/ LLM補完 | **★最高誤抽出リスク** |
| 折れ線グラフ（PKプロファイル） | 高 | 図番号+キャプション抽出+Vision-LLM | グラフ番号とテキスト参照の不一致 |
| 散布図・棒グラフ（有効性） | 中 | 図キャプション抽出+参照テキスト照合 | — |
| 化学構造式 | 低 | 画像メタデータ保持 | 構造式は画像として保持 |
| 数式・統計式 | 低 | 画像保持 | — |
| 脚注・凡例 | 高 | 脚注とメイン表の関連付け | **★脚注分離が誤抽出の主要因** |

### 4.2 テキスト-表-グラフ間の参照関係維持

CTD文書における参照関係パターン（3種類）:

- **Type A: テキスト→表参照** 「PKパラメータを表1に示す」→ テキストチャンクに表1の数値が必須
- **Type B: テキスト→図参照** 「血中濃度推移を図2に示す」→ テキストチャンクに図2のキャプション・数値が必須
- **Type C: 表→図参照** 「図3に示す溶出プロファイルの数値を表2に示す」→ 表と図の双方向リンクが必要

> ★ 参照関係が切断されると、IF文書中で「表2参照」と記載しながら実際の数値が不正確になる致命的誤りが生じる

## 5. IF品質評価メトリクス設計

### 5.1 評価メトリクス体系（8メトリクス + Composite）

| 評価カテゴリ | 評価メトリクス | 算出方法 | 重み（初期） |
|------------|--------------|---------|------------|
| テキスト類似度 | ROUGE-1/2/L | N-gram重複率 | 10% |
| 意味的類似度 | BERTScore F1（multilingual） | BERT埋め込みコサイン類似度 | 15% |
| **医薬数値一致率** | 数値・単位一致率 | 正規表現数値抽出+差分 | **25%** |
| 表構造再現率 | 表の行列構造一致率 | 表抽出後Jaccard係数 | 15% |
| 図表参照整合性 | 図番号・表番号参照一致率 | 参照番号抽出+照合 | 10% |
| セクション構造準拠 | Ⅰ〜ⅩⅢ全セクション存在・順序 | 構造チェックリスト | 10% |
| 専門用語カバレッジ | IFセクション別専門用語出現率 | 医薬辞書照合 | 5% |
| 引用文献一致率 | 引用文献Jaccard係数 | 文献リスト差分 | 5% |
| 表紙完備率 | 表紙20項目完備率 | 必須項目存在チェック | 5% |

```python
# 複合スコア計算
CompositeScore = Σ(weight_i × score_i)

weights = {
    "text_similarity":     0.10,  # ROUGE-L
    "semantic_similarity": 0.15,  # BERTScore F1
    "numerical_accuracy":  0.25,  # 最重要 — 医学的正確性
    "table_reproduction":  0.15,  # 表構造再現
    "figure_ref_match":    0.10,  # 図表参照整合
    "section_coverage":    0.10,  # Ⅰ〜ⅩⅢ準拠
    "term_coverage":       0.05,  # 専門用語
    "citation_match":      0.05,  # 文献
    "cover_completeness":  0.05,  # 表紙20項目
}
```
